## 📊 Physics Metrics Explained

### 1️⃣ **Throat Radius (a) = 1.5 M**
```
Definition: Minimum wormhole radius
Unit: Schwarzschild mass units (M☉)
Role: Fundamental wormhole parameter
Display: Always "1.50 M"
```

### 2️⃣ **Spacetime Curvature (R)**
```
Equation: R = 4/a²
Meaning: How much spacetime is bent
Levels:
  - High: R > 1 (Significant warping)
  - Very High: R > 2 (Extreme warping)
  - Moderate: R ≤ 1 (Gentle curve)
Source: Ricci curvature tensor
```

### 3️⃣ **Time Dilation (α)**
```
Equation: α = √(1 - 2M/r)
Range: 0.0 to 1.0
Meaning: Time passes slower near the wormhole
Example: 1 hour near throat = 1.37 hours far away
Physical Effect: From Einstein's General Relativity
```

### 4️⃣ **Tidal Forces (F)**
```
Equation: F ∝ M/r³
Safety Levels:
  🟢 Safe: F < 0.2 (No danger)
  🟡 Moderate: 0.2 ≤ F < 0.5 (Uncomfortable)
  🔴 High: F ≥ 0.5 (Dangerous - "spaghettification")
Meaning: Differential gravitational pull on objects
```

### 5️⃣ **Render Performance (FPS)**
```
Calculation: 60-frame moving average
Color Coding:
  🟢 Green: 60 FPS (Optimal)
  🟡 Yellow: 55-59 FPS (Good)
  🔴 Red: < 45 FPS (Warning)
Purpose: Monitor real-time performance
```
