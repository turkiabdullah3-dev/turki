Project Overview

Spacetime Observatory is an interactive educational experience exploring the physics of extreme spacetime geometries.

The project features two main modes:

  • Black Hole Mode: Explore Schwarzschild spacetime with accurate time dilation, gravitational redshift, and tidal forces
  • Wormhole Mode: Traverse a Morris-Thorne traversable wormhole with exotic matter visualization

All physics equations are scientifically accurate based on Einstein's General Relativity, with safety mechanisms to ensure stable numerical computation.

Technical Implementation

Built with modern web technologies:

  • Canvas 2D rendering for optimal performance
  • Adaptive quality system (works on laptop and iPad)
  • Real-time physics simulation with safety clamps
  • KaTeX for mathematical equation rendering
  • Liquid glass UI design with procedural space backgrounds
  • CSP security headers and XSS protection

Scientific References

This project is based on established physics literature:

  • Schwarzschild Metric: K. Schwarzschild (1916) - First exact solution to Einstein's field equations
  • Morris-Thorne Wormhole: Morris & Thorne (1988) - "Wormholes in spacetime and their use for interstellar travel"
  • General Relativity: A. Einstein (1915) - Foundation of modern gravitational physics

Note: This is an educational visualization, not a research-grade simulation. For accurate scientific work, use specialized numerical relativity software.

Disclaimers

  • No personal data is collected or transmitted. All settings are stored locally in your browser.
